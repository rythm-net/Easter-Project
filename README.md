## The Easter Bunny game build entirely on Java

### Пловдивски Университет - Шаблони за проектиране (2023) - Задача за оценка по упражнения

#### Подробности за заданието:
Великденският заек имал достъп до вълшебна торба, която можела да създава три вида яйца: пъдпъдъчи, кокоши и динозавърски. Когато дадено яйце било създадено, той го взимал и оцветявал или облепял със стикери на произволен принцип. Украсените яйца поставял в кошница. Имало 3 малки пиленца, които наблюдавали кошницата и когато яйцата в нея станели повече от 5 броя, те в един глас съобщавали: “Яйцата са готови”.



Опишете програмно процесът на приготвяне на великденски яйца. За целта използвайте шаблоните: Singleton, Factory, Decorator, Observer

Великденският заек може да бъде само един. Factory може по избор да е Factory method или Abstract Factory.
